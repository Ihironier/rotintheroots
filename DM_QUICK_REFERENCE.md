# DM Quick Reference

This is a quick reference guide for Dungeon Masters running the "The Rot in the Roots" campaign. It includes essential details about the campaign and shorthand for common rules.

## Key Notes
- **Campaign Theme:** Horror Fantasy
- **Setting:** Dark forest and ruins
- **Suggested Levels:** 1-10
- **Important Locations:** Hollowed Grove, Sunken Ruins, The Withering Tower

## Rules Reference
- **Rolling for Initiative:** Roll a d20 + Dexterity Modifier
- **Failure Threshold:** 10 for most skill checks

Keep this handy while running your sessions!
