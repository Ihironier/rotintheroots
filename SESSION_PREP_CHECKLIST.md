# Session Prep Checklist

- [ ] Review campaign goals and plot points
- [ ] Prepare character sheets and player handouts
- [ ] Create necessary NPC stat blocks
- [ ] Outline session objectives
- [ ] Gather materials and props needed for the session
- [ ] Set up digital tools (e.g., Roll20, D&D Beyond)
- [ ] Prepare any visual aids or maps
- [ ] Confirm availability of players for the session
- [ ] Prepare refreshments and snacks (optional)
